Privacy Policy – ASA Bridge Extensions

Last updated: 2026-04-08

This privacy policy applies to the following Chrome extensions:

  • ASA Bridge for ChatGPT
  • ASA Bridge for Gemini
  • ASA Bridge for Claude

What data is accessed

Each extension reads the conversation content displayed on the respective AI service page (ChatGPT, Gemini, or Claude) using standard browser APIs. This data is used solely to relay conversation snapshots to the companion desktop application (ASA) running on your local PC.

How data is used

The conversation data is sent exclusively to http://127.0.0.1 (localhost) — the same machine your browser is running on. It is never transmitted to any external server, third party, or cloud service.

Data storage

The extension does not store conversation data persistently. Data is passed in memory to the local desktop application and is not retained by the extension itself.

Permissions

The extensions request the following permissions:

  • scripting – to read conversation content from the page
  • tabs – to identify the active conversation tab
  • storage – to store minimal operational state (e.g. connection settings) locally in the browser

Contact

If you have questions about this policy, please open an issue on the relevant GitHub repository.
